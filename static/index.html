How to build a custom HID report

Format: 01:MM:00:KK:00:00:00:00:00

  • MM = modifier byte (OR the values): Ctrl 01, Shift 02, Alt 04, GUI/Win 08
  • KK = key usage code (examples: Tab 2B, Enter 28, C 06, V 19, F4 3D)
  • Examples: Alt+Tab = 01:04:00:2B:00:00:00:00:00, Ctrl+C = 01:01:00:06:00:00:00:00:00
How to find the HEX code for a key
  • Use the USB HID Keyboard/Keypad Usage Table (Usage Page 0x07) and find your key name.
  • Use the key's usage ID as KK in hex (2 digits). Example: key Q has usage 0x14, so KK is 14.
  • For letters, a quick rule is: A=04, B=05, ... Z=1D. Numbers are 1=1E through 0=27.
  • Then combine modifiers in MM (for example Ctrl+Shift = 01|02 = 03) and build the final report.
